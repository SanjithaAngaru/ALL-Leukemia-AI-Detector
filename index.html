Acute Lymphoblastic Leukemia (ALL) Detection Tool

This project uses a machine learning model trained with Google Teachable Machine to help classify blood smear images as either Healthy or showing signs of Acute Lymphoblastic Leukemia (ALL).

This model was created for a science fair project and demonstrated approximately 75% accuracy when tested on unseen images. It shows how artificial intelligence can support early screening in medical imaging.

⚠ This tool is NOT a medical diagnostic device. It is for educational and experimental purposes only.

You can choose between two detection methods:

  • Webcam Detection – Uses your camera to classify live images.
  • Image Upload Detection – Upload a blood smear image for analysis.
